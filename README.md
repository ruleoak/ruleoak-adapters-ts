# @ruleoak/adapters-ts v1.0.0

TypeScript adapters for adding RuleOak permission, approval, evidence, and replay to agent frameworks and tool-calling apps.

## Use this repo when you need

- generic TypeScript tool wrappers
- MCP-style tool adapter patterns
- OpenAI Agents JS-style wrapper patterns
- LangChain.js-style wrapper patterns
- Vercel AI SDK-style wrapper patterns
- coding-agent command wrappers
- OpenClaw-style action normalization

All framework integrations are optional and tests use local mocks.

## Public repository

- GitHub: https://github.com/ruleoak/ruleoak-adapters-ts
- Version: `1.0.0`
- License: `Apache-2.0`

## Relationship to RuleOak Core

This repo provides adapter interfaces and wrappers. RuleOak Core remains the full Agent Firewall + Flight Recorder runtime and is licensed under `AGPL-3.0-or-later` with commercial licensing available.

npm install
